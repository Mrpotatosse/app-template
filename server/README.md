# server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.30. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

# init the project

```bash
# call this 1 time
bun run init
```

# starting dev

```bash
bun run dev
```
